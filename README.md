# CorporateFlow Automation
Criar uma aplicação Python capaz de automatizar fluxos corporativos baseados em arquivos, planilhas e regras de negócio, com foco em:  leitura de arquivos; validação; normalização; processamento; geração de logs; backup; tratamento de erros; relatórios finais.  A ideia é mostrar que você sabe construir uma solução escalável.
